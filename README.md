# 🎯 Catch Me Game

A browser-based JavaScript reflex game where the player must click a moving button before time runs out.

---

## 📖 About

Catch Me Game is one of my portfolio projects created while learning JavaScript.

The objective is simple:
- Choose a difficulty.
- Click the moving button as many times as possible.
- The button becomes smaller and faster as your score increases.
- Try to beat your High Score.

---

## ✨ Features

- Three difficulty modes
  - Easy
  - Medium
  - Hard

- Individual High Score for every difficulty

- High Scores saved with Local Storage

- Dynamic difficulty system

- Progressive button shrinking

- Progressive button speed increase

- Sound effects

- Random button movement

- Game Over screen

- Play Again

- Back to Menu

- Responsive layout
